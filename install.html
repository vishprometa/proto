What this does:

  • Downloads the appropriate binary for your system (macOS ARM64/x64, Linux ARM64/x64)
  • Installs Proto to /usr/local/bin/proto
  • Makes it available as a global command
  • Handles all dependencies automatically

After Installation:

  1. Run proto to start the interactive chat
  2. Follow the onboarding to configure your ClickHouse connection
  3. Enjoy the built-in AI model (no API keys needed)
  4. Start asking questions about your data!

Alternative Installation Methods

Using pipx (Recommended for Python users)

pipx install proto-clickhouse-agent

Using pip

pip install proto-clickhouse-agent

Manual Download

Download the appropriate binary for your system:

  • macOS ARM64
  • macOS x64
  • Linux ARM64
  • Linux x64

System Requirements

  • macOS 10.15+ or Linux
  • ClickHouse database (local or cloud)
  • AI provider (Local LLM built-in)
  • ~3.5GB free space for AI model (first run)

Getting Help

  • Run proto --help for command help
  • Visit Proto website for documentation
  • Check support page for help
